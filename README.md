# vuehelloworld

This is a test application for creating a website UI and API calls. It will generate a random pokemon with random moves for it based off what it can learn.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
